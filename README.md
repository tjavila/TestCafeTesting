# TestCafeTesting
Project created to practice end-to-end test automation using the TestCafe framework and a fake bank API.

## Dependencies
- TestCafe 3.0.1
- Prettier 3.0.0

## Run tests:
Install dependencies with the command:
```
npm i
```
Run some script from the package.json file using 
```
npm run <script_name>
```
An example:
```
npm run test:chrome
```
